# Production Environment Variables

All required `.env` keys for `sinfat.com` (Oracle Cloud VM).

Location on server: `/var/www/sinfat/.env`

---

## App

| Key | Value | Notes |
|-----|-------|-------|
| `APP_NAME` | `sinfat` | |
| `APP_ENV` | `production` | |
| `APP_DEBUG` | `false` | **Never** `true` in production |
| `APP_URL` | `https://sinfat.com` | Must include `https://` |
| `APP_KEY` | `base64:...` | Generated by `php artisan key:generate` |

## Database

| Key | Value | Notes |
|-----|-------|-------|
| `DB_CONNECTION` | `mysql` | |
| `DB_HOST` | `127.0.0.1` | |
| `DB_PORT` | `3306` | |
| `DB_DATABASE` | `sinfat_portfolio` | |
| `DB_USERNAME` | `sinfat` | |
| `DB_PASSWORD` | *(secret)* | Set during MySQL setup |

## Cache, Session & Queue

| Key | Value | Notes |
|-----|-------|-------|
| `CACHE_DRIVER` | `redis` | Redis is installed and active on the VM |
| `SESSION_DRIVER` | `redis` | Faster than `database` |
| `QUEUE_CONNECTION` | `redis` | Needed for queued jobs |
| `REDIS_HOST` | `127.0.0.1` | |
| `REDIS_PORT` | `6379` | |

## AI Integration

| Key | Value | Notes |
|-----|-------|-------|
| `AI_PROVIDER` | `github` | Options: `github`, `anthropic`, `gemini`, `ollama` |
| `GITHUB_MODELS_KEY` | `github_pat_...` | Required if provider is `github`. Free via GitHub PAT with Models permission |
| `GITHUB_MODELS_MODEL` | `gpt-4o-mini` | Default model for GitHub Models |
| `ANTHROPIC_API_KEY` | `sk-ant-...` | Required if provider is `anthropic` |
| `ANTHROPIC_MODEL` | `claude-sonnet-4-5` | Or any supported model |
| `GEMINI_API_KEY` | *(optional)* | Required if provider is `gemini` |
| `GEMINI_MODEL` | *(optional)* | e.g. `gemini-2.0-flash` |

| Key | Value | Notes |

---

## How to Edit

```bash
ssh -i ~/.ssh/sinfat-portfolio.key ubuntu@<server-ip>
sudo nano /var/www/sinfat/.env
cd /var/www/sinfat && php artisan config:cache
```

Always run `config:cache` after editing `.env` — Laravel reads from the cache in production.
